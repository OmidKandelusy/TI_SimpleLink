# Description
This repository illustrates how the simplelink sdk by Texas Instruments can be used to implement embedded projects. In specific, this project implements a shell over subGHz wireless communication where two devices can send commands to each other over the wireless link. The user can configure the wireless connection for a specific frequency band. This project clearly shows: 

a) how to use the TI RTOS threads and features  for real time applications.

b) how main peripherals like UART, Radio Core, GPIO, and timers can be used in projects.

c) how sysconfig and ecplise build environment are used in the simplelink build ecosystem.


This is a work in progress and this readme file will be updated soon.
